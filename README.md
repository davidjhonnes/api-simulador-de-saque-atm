## Description
API Saque notas e transações GSM utilizando:

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app
(!) Important! RUN SEED:UP

```bash
# development
$ yarn start

# VERY IMPORTANT seed up to insert data and users
$ yarn seed:up

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod

# clear database
$ yarn seed:down

# restore database
$ yarn seed:up
```
